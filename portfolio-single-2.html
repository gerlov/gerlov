Sudoku Game

Project Date October 1th - 25th, 2023
Teachings JavaFX, MVC, GUI
Project School Project
Code Github - Sudoku

Project Background:

Leveraging my proficiency in Java and JavaFX, I developed a fully functional Sudoku game. This project was structured using the Model-View-Controller (MVC) pattern, demonstrating a clear separation of concerns and enhancing the maintainability of the code.

Technical Implementation:

The objective was to create an engaging and interactive Sudoku game that users could play on their computers. The project's key components included:

Java: Utilized for the game's logic and backend functionalities, ensuring a solid and reliable performance.

JavaFX: Chosen for the front-end to provide a rich user interface, making the game visually appealing and user-friendly.

MVC Architecture: Applied to organize the project, facilitating ease of development and future scalability. This approach separated the game's logic (Model), the graphical interface (View), and the control flow (Controller), promoting a clean and modular codebase.

Conclusion:

This Sudoku game project highlights my capability to design and implement sophisticated software applications using advanced programming languages and frameworks. It showcases my skills in creating interactive applications with a focus on user experience and software architecture.
