Project Background:

This project, "Labyrinth of the Minotaur," was developed by a group of students at KTH Royal Institute of Technology. The game is a network-based multiplayer experience for 2-6 players, featuring roles such as hunter and humans. The objective is to either escape the labyrinth or catch all the humans.

Project Overview:

Game Mechanics: The game utilizes SDL for graphics, UDP for fast-paced network communication, and TCP for reliable connections. The labyrinth is randomly generated for each session, providing a unique experience every time.

Roles and Objectives: Players are assigned either the role of hunter or humans. The humans must navigate the labyrinth to find the exit while avoiding the hunter, who aims to capture them.

Technical Implementation: The game was developed using the C programming language, leveraging the SDL library for handling graphical rendering and user input. Network communication was managed through UDP to ensure fast gameplay.

Conclusion:

This project demonstrated our ability to develop a complex multiplayer game using various technologies and methodologies. It highlights our proficiency in C programming, network communication, and game development, showcasing our teamwork and problem-solving skills.
